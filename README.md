## Getting Started

From pypi:

```
pip install oki
```

To hack on the code, setup a virtual environment, grab the code, and then:

```
# install dependencies
make install

# run the tests
make test

# review the code for style, etc.
make review

# run a test coverage report
make coverage

# check for other commands available
make list
```
